## Sistema de Tokens

- **Duraci√≥n**: 15 minutos
- **Prop√≥sito**: Autorizar peticiones a la API

### **Refresh Token (JWT)**

- **Duraci√≥n**: 7 d√≠as
- **Prop√≥sito**: Renovar access tokens

### **State Token (OAuth)**

- **Duraci√≥n**: 10 minutos
- **Prop√≥sito**: Protecci√≥n CSRF en OAuth
- **Almacenamiento**: Cookie HTTPOnly temporal
